# MagangTrack

Sistem absensi magang berbasis web dengan validasi GPS dan kamera real-time.

## Struktur Proyek
- `docs/`: Dokumentasi teknis dan PRD.
- `frontend/`: React + Vite + TS (TBD).
- `backend/`: Express.js (TBD).

## Tech Stack
- Frontend: React + Vite + TS + Tailwind
- Backend: Express.js
- Database: Neon (PostgreSQL)
