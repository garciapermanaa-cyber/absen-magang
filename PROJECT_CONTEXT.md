# Project Context - MagangTrack

## Masalah
Intern sering terlambat atau pulang awal tanpa catatan yang andal. Sistem manual/chat mudah dimanipulasi.

## Solusi
Aplikasi web yang memaksa penggunaan kamera (lokal) dan lokasi GPS (validasi backend) untuk check-in/out.

## Prinsip Utama
1. **Privacy First**: Foto tidak boleh di-upload atau disimpan.
2. **Server Authority**: Waktu dan validasi lokasi harus dari server.
3. **Lightweight**: Frontend cepat dan responsif.
