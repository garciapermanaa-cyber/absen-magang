# MagangTrack

## Description
MagangTrack is a web-based attendance system for interns designed to provide a structured and reliable way to track check-in and check-out times using camera activation and GPS validation.

## Current Project Phase
Documentation / PRD

## Tech Stack Summary
- **Frontend:** React, Vite, TypeScript, Tailwind CSS
- **Backend:** Express.js
- **Database:** Supabase PostgreSQL
- **Deployment:** Vercel
- **Browser APIs:** Camera API (getUserMedia), Geolocation API
